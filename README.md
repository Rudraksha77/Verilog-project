# Verilog Projects

## 📖 Overview
This repository contains my Verilog HDL projects covering both **Combinational** and **Sequential Logic Circuits**. These projects are created for learning, practice, and digital design implementation. Each project includes well-structured Verilog code and a corresponding testbench for simulation and verification.

## 📂 Repository Contents

### Combinational Circuits
- Half Adder
- Full Adder
- Comparator
- Decoder
- Encoder
- Priority Encoder
- Multiplexer (MUX)
- Demultiplexer (DEMUX)

### Sequential Circuits
- Left Shift Register
- Right Shift Register
- Memory
- Flip-Flops (Coming Soon)
- Counters (Coming Soon)
- Registers (Coming Soon)

## 🎯 Purpose
- Practice Verilog HDL programming.
- Learn digital logic design concepts.
- Verify designs using simulation testbenches.
- Build a collection of reusable Verilog modules.
